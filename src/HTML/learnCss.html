Grouping CSS selectors can be a very handy trick. This option allows for the same type of style to be applied to multiple selectors without retyping the statement. Below is the syntax for grouping selectors.
/*propertyName: value would be applied to all the selectors listed*/ .cssSelector1, .cssSelector2, .cssSelector3, #id, HTMLElementSelector{ propertyName: value; }
Now for a real example!
<!--In another HTML document--> <p class="dummyClass">Dummy text for a dummy class...</p> <p id="dummyId">Dummy text for a dummy id...</p> <p id="dummyId2">Dummy text for another dummy id...</p>
/*In a separate CSS file*/ /*only the class, dummyClass, and the id, dummyId2 will be affected by the below CSS statement*/ .dummyClass, #dummyId2{ color: pink; }

Context dependant selectors... order matters. Do you remember nested lists from the HTML section? This is similar to a point. Context dependency is used to describe styling that only appears if the particular element is located in a specific location or situation. You can style just the anchor tags in a navigation bar. Other anchor tags in the page will not follow the styling unless they are children of your navigation bar.
<!--In another HTML document--> <div class="dummyClass">Dummy text for a dummy class... However, this time context dependent selectors are being demonstrated. Starting a new paragraph....<p>NOW! Pseudo-classes describe a specific and unique state of an HTML element. They are often used to style links to other pages. A link may change colors after being visited. Additionally, images may increase in size slightly when the user's mouse is hovering over the given image.
<!--In another HTML document--> <div class="box"><img src="http://www.icon100.com/up/3772/128/425-hand-pointer.png"/></div> <div class="secretMsg">Boo!</div>
/*In a separate CSS file*/ /*Will render a green box*/ .box{ display: block; width: 128px; height: 128px; background-color: greenyellow; } /*Styling for the message*/ .secretMsg{ font-size: x-large; color: turquoise; display: none; /*hides the message*/ text-align: right; } /*While hovering over the box, show the message*/ /*.box:hover is the pseudo-class + .secretMsg specifies that the secret message is the element to style.*/ .box:hover + .secretMsg{ display: block; /*reveals the message*/ }
